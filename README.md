# C++ Systems Design

Advanced systems programming and distributed systems implementations -
This commit expands the repository with a broad collection of C++ implementations covering:

* Concurrency primitives and synchronization mechanisms

  * Blocking queues
  * Circular buffers
  * Counting semaphores
  * Reader-writer locks
  * Cyclic barriers
  * Thread pools
  * Task schedulers
  * Lock-free ring buffers
  * Atomic counters using CAS

* Caching and storage-related components

  * LRU caches
  * Concurrent hash maps
  * Redis-style cache with TTL expiration
  * MemTable and SSTable lookup structures

* Distributed systems concepts

  * Raft leader election
  * Consistent hashing
  * Asynchronous logging
  * Multi-threaded web crawler

* Rate limiting algorithms

  * Token bucket
  * Sliding window

* Large-scale data processing

  * External sort
  * Merge K sorted files
  * Top-K IP detection in massive log files
  * Duplicate detection in large datasets
  * Streaming reads from very large files

* System simulations and coordination problems

  * Elevator scheduling
  * Traffic light controller
  * Dining philosophers
  * H2O formation
  * Ordered thread execution patterns

These implementations were developed to deepen understanding of backend infrastructure, storage engines, concurrency control, distributed systems, and large-scale data processing.
